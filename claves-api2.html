Ahora es el momento de obtener el id del cliente de OAuth.

Vamos a la pantalla de consentimiento de OAuth.


Seleccionamos externos para que nos funcione en la Raspberry Pi.


Rellenamos lo que nos piden con nuestros datos personales.


Ya podemos crear la credencial de OAuth.

Ahora toca poner nuestra clave API y nuestro cliente ID de OAuth en nuestra Raspberry para ver YouTube.

En la Raspberry vamos a addons de video y entramos en Youtube, vamos a la configuración.


Ahora vamos a API.


Una vez estamos en API rellenamos con los datos anteriores, API Key, API ID y API Secret, le damos a OK y ya nos funcionará YouTube.
