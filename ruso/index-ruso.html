Raspberry — это недорогой компьютер компактного формата, предназначенный для того, чтобы сделать вычисления доступными для всех пользователей. Raspberry Pi также широко используется для разработки небольших прототипов и для обучения компьютерам и электронике в школах.
Raspberry Pi — это простая компьютерная плата, состоящая из SoC, ЦП, ОЗУ, портов ввода и вывода аудио и видео, сетевого подключения, слота SD для хранения, разъема питания и разъемов для периферийных устройств.
