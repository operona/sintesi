LibreELEC es un sistema operativo basado en el kernel de Linux. Es un sistema operativo diseñado para cargar solo el software suficiente para permitir que un ordenador ejecute Kodi.

La principal ventaja de usar LibreELEC es que es una forma más eficiente de ejecutar Kodi en comparación con el uso de un sistema operativo más robusto como Windows o Mac OS. Debido a esto, generalmente puede ejecutar LibreELEC en un HTPC menos potente sin sacrificar la imagen o el sonido y el dinero para un equipo más potente.


Un add-on es un módulo adicional que complementa navegadores, programas de correo y otras aplicaciones de software y hardware y añade funciones que el sistema básico no tiene.

Kodi tiene add-ons para añadir música, vídeo, streaming, subtítulos, letras de canciones, juegos, y mucho más.
